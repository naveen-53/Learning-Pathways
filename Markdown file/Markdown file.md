# Markdown file Reference

This file is a reusable reference for writing markdown files.

---

## Headings
- # Heading 1
```
# heading 1 name
```
- ## Heading 2
```
## heading 2 name
```
- ### Heading 3
```
### heading 3 name
```
- #### Heading 4
```
#### heading 4 name
```
- ##### Heading 5
```
##### heading 5 name
```
- ###### Heading 6
```
###### heading 6 name
```
---

## Text Formatting
- Bold
```
**text**
```
-Italic
```
*text*
```
- Bold and Italic
```
***text***
```
- Strike through text
```
~~text~~
```

---

## Lists

### Unordered List
 
```
- item 1
- item 2
  - sub item 1
  - sub item 2
```
##### Example :
- Item One
- Item Two
  - Sub Item
  - Sub Item

### Ordered List
```
1. First
2. Second
3. Third
```
##### Example :
1. First
2. Second
3. Third


---

## Links

```
[link name](link)
```
##### Example :
[Learning pathway](https://github.com/naveen-53/Learning-Pathways)

---

## Images

```
![Alt Text](image-url.png)
```
##### Example :
![sample image](https://github.com/naveen-53/Learning-Pathways/blob/main/Images/naveen%20logo%201.jpg)

## Inline Code
```
text `inline code` text
```

##### Example :
Use `git status` to check status

## Table
```
| column 1 | column 2 | column 3 |
|-----|------|--------|
| value 1 | value 2 | value 3 |
```
##### Example :
| Name | Tech | Status |
|-----|------|--------|
| App | Angular | Done |
| API | Spring | In Progress |

## Task List
```
- [x] checked box
- [ ] unchecked box
```
##### Example :
- [x] Setup project
- [ ] Write README
- [ ] Deploy
